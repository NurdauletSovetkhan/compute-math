# Week 1 — Assignment 1: Численные методы (поиск корней)

Этот раздел содержит 10 учебных заданий по численным методам решения уравнений одного переменного. В каждом задании демонстрируются классические алгоритмы (бисекция, ложная позиция, простая итерация, Ньютон, секущие, Мюллер), сравнение их сходимости, а также критерии остановки.

## Состав папки

- `task1.py` — графический анализ уравнения tan(x) = x^2 на отрезке [0.5, 1.5], сохранение графика `task1_plot.png` и численная проверка знака f(x) = tan(x) − x^2.
- `task2.py` — метод бисекции для двух функций:
  - f1(x) = tan(x) − x^2 на [0.5, 1.5]
  - f2(x) = x^3 − 2x − 5 на [2, 3]
  Выводится итерационная таблица приближений корня для каждой функции.
- `task3.py` — метод простой итерации для уравнения ln(x+1) + x^2 = 0, анализ условий сходимости и демонстрация итераций для выбранного отображения g(x) (используется g(x) = e^{−x^2} − 1), вывод таблицы итераций и значения функции в финальной точке.
- `task4.py` — метод Ньютона для f(x) = 5x − cos(x) − 3. Выводится подробная таблица (через `tabulate`) с шагами итераций и критерием остановки по |Δx|.
- `task5.py` — метод секущих для уравнения e^{−x} = x (эквивалентно f(x) = e^{−x} − x = 0) на [0, 1]. Печать таблицы итераций и финальной аппроксимации.
- `task6.py` — метод ложного положения (Regula Falsi) для f(x) = x^3 − 3x + 1 на [0, 1]. Запускается ровно 3 итерации с печатью промежуточных значений.
- `task7.py` — сравнение методов Мюллера, Ньютона и секущих для полинома f(x) = x^4 − 3x^3 + x^2 + x + 1:
  - Поиск (возможных комплексных) корней с разными начальными приближениями
  - Дедупликация найденных корней по допуску
  - Замер времени выполнения и сравнительная сводка
- `task8.py` — сравнение сходимости пяти методов (бисекция, ложная позиция, простая итерация, Ньютон, секущие) на f(x) = x^3 − 2x − 5. Строится и сохраняется график `convergence_comparison.png` с траекторией x_i и логарифмической шкалой ошибки |x_i − x*|.
- `task9.py` — сравнение критериев останова по абсолютной (Ea) и относительной (Er) погрешности для секущих и Ньютона на нескольких примерах; печать кратких сводок и подробной таблицы шагов (на примере e^{−x} − x).
- `task10.py` — сводная таблица x_i и f(x_i) для нескольких методов, на примере f(x) = ln(x) + x, g(x) = e^{−x}. Сводка строится для: бисекции, ложной позиции, простых итераций, Ньютона и секущих, включая стартовые значения.
- `methods/` — реализации численных методов:
  - `bisection.py` — метод бисекции: `bisection(func, a, b, epsilon, max_iterations)` → список кортежей `(k, c, f(c))`;
  - `false_point.py` — метод ложной позиции: `false_point(func, a, b, epsilon, max_iterations)` → список `(k, c, f(c))`;
  - `fixed_point.py` — простая итерация: `fixed_point(g, x0, eps, max_iter)` → список `(k, x_k, x_{k+1}, |Δx|)`;
  - `newton_raphson.py` — метод Ньютона: `newton(func, dfunc, x0, epsilon, max_iterations)` → список `(k, x_k, x_{k+1}, f(x_k))`;
  - `secant.py` — метод секущих: `secant(func, x0, x1, epsilon, max_iterations)` → список `(k, x_k, x_{k+1}, f(x_k))`;
  - `muller.py` — метод Мюллера с поддержкой комплексных чисел: `muller(func, x0, x1, x2, epsilon, max_iterations)` → список `(k, x_k, x_{k+1}, f(x_k))`.

## Как запускать

Из папки `week-1/assignment-1` выполните нужный скрипт:

```bash
# Примеры
python task1.py   # сохранит график task1_plot.png и выведет сводку по f(x)
python task2.py   # сравнение бисекции на двух задачах

python task5.py   # Секущие для e^{-x} = x на [0,1]
```

Выходные PNG-файлы:

- `task1_plot.png` — графический анализ tan(x) и x^2 на [0.5, 1.5]
- `convergence_comparison.png` — график сходимости методов для x^3 − 2x − 5

## Краткие подсказки по параметрам

- Начальные приближения и интервалы задаются прямо в соответствующих `task*.py`:
  - Бисекция/ложная позиция: параметры `a`, `b`
  - Простая итерация: функция отображения `g(x)`, начальное `x0`, допуск `eps`
  - Ньютон: начальное `x0`, производная `df(x)`
  - Секущие: `x0`, `x1`
- Критерии остановки: `epsilon`/`tol` (обычно по шагу |x_{k+1} − x_k|, иногда также по |f(x)|)
- Для комплексных корней (метод Мюллера, `task7.py`) начальные значения могут быть комплексными: `x0 = 1+1j` и т.п.

## Мини-контракты (что возвращают методы)

- Bisection/False Position: список `(k, c, f(c))`, где `c` — текущая аппроксимация корня.
- Fixed-Point: `(k, x_k, x_{k+1}, |Δx|)`.
- Newton/Secant: `(k, x_k, x_{k+1}, f(x_k))`.
- Muller: `(k, x_k, x_{k+1}, f(x_k))`, поддерживает `complex`.

Это удобно для построения единых таблиц и графиков сходимости (см. `task8.py`, `task10.py`).

## Полезные замечания

- Для устойчивой простой итерации требуется отображение g(x) с |g'(x)| < 1 на рассматриваемом отрезке (см. комментарии в `task3.py`).
- При сравнении абсолютной и относительной погрешности (см. `task9.py`) будьте осторожны в окрестности нулевых корней — относительная ошибка может вводить в заблуждение.
- Если `tabulate` не установлен, часть скриптов содержит простой fallback формат вывода.
